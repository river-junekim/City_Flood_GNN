## ⚠️ 현재 진행 상태 (2026-06-19) — 프로젝트 재정의 중 (이 섹션 먼저 읽을 것)

> 아래 §1~16은 원래 "GNN 5모델 비교" 프로젝트 설명(데이터·구조 참고용). 하지만 EDA 결과 **프로젝트가 근본적으로 재정의되는 중**이다. 내일은 여기서 이어간다.

### 무슨 일이 있었나 (데이터 품질 조사 결과)
- `flood_flag = level>0` 라벨이 **심각하게 오염** — 도로 "1cm"는 대부분 baseline 노이즈(연중 8.3%, 마른 1월에도 56%). 진짜 침수는 극희소.
- **도로 = 이산 단계 게이지** (0,1,2,3,4,6,11,...,96). `96`=측정범위 천장 saturation 오류, `999/1000/312/419`=에러코드.
- **하수 = 연속 기저유량** (95% >0), surcharge 드묾.
- **변환·처리는 정상** (원본 CSV=parquet 2회 정확 검증). 문제는 라벨 정의 + 센서 아티팩트.
- 베이스라인 모델: **강우(공공, 48관측소, 도로와 1.45km)가 침수를 예측 못함**(평평한 강우-침수율). 단 AWS(촘촘) 추가 시 달라질 수 있음 — **미확정**.
- **하수→도로 선행성은 미확정** (과거 분석은 오염라벨·거리매칭 기반). **단정 금지, 검증으로 판단.**

### 현재 방향: 시계열 전처리 재설계
- **반드시 읽을 문서**: `TIMESERIES_PREPROCESSING_DESIGN.md`(최신 설계 ★), `DATA_QUALITY_FINDINGS.md`, `PREPROCESSING_PLAN_FROM_EDA.md`, `eda_road_sewer_report.html`, `eda_raw_report.html`.
- **확정 결정**: nowcast(타겟 t+H, H=1 → forecast 확장) / 10분 해상도 / **결측 마스크 중심** / **T_in=6스텝** / **차분(level_diff) 피처 추가** / **persistence 베이스라인 능가**로 평가 / **도로·하수 도메인 분리 후 조인**.
- **stuck-run 임계**: run-length 분포로 **≥1day** (초기 추측 60분/6시간은 폐기). level≥6/고수위 한정 재분석 남음.
- **라벨**: `level≥T AND 오류·saturation 제외` (T=6 유력, 고객 운영기준 미확정 → T=2/6/10 비교).

### 참고 (Claude 의견 — 표시용, 단정 아님)  ※ 2026-06-19 강우필요성 검토로 정정
- **이전 "전부 강우 전 잠정" 프레이밍은 과했음(정정).** 검토 결과(노트북 §7):
  - 도로↔하수 차분상관이 **여름·겨울 모두 ~0**(0.036/-0.004). 강우 매개면 여름≫겨울이어야 하나 아님 → **약한 결합은 강우 부재 아티팩트가 아니라 실제로 약함**(강우 없이도 견고).
  - 하수 급상승(물리신호) 트리거로 라벨을 우회해도 도로 반응 약함(동시·사후/사전 1.6배). → 모든 슬라이스에서 약함.
- **그래도 강우가 필요한 곳은 한정적**: ① 도로 라벨 겨울오염 정제(도로수위 평균 여름0.45<겨울0.88 역전 = 오염 확증, flood_t6 12월 최고), ② 교란요인 통제(하수가 강우 이상으로 도로 예측 기여하는지), ③ 실제 호우 regime 분리.
  → 즉 강우는 '숨은 강한 관계'를 드러내는 용도가 아니라 **라벨 정제·인과/운영 해석용**. 약한 결합 결론은 유지하되 **null 확정은 강우 확보까지 보류**.

### 작업 규율 (중요 — 사용자 피드백 반영)
- **추측·단정 금지.** 임계·관계는 **데이터로 검증해서** 정한다 (예: 60분/6시간 추측 → run 분포로 ≥1day 수정).
- **과장 금지.** "99% 가짜"는 해상도+임계+강우를 섞은 부정확 표현이었음(정정함). 확정 vs 미확정을 구분해 보고.
- 강우는 **고객 제공이 아닌 공공데이터** — 두 센서(도로·하수)만으로 우선 진행, AWS는 나중에.
- 데이터 직접 접근은 이번 EDA 작업 한정 허용됨. **EDA raw 데이터 내용은 영구 메모리에 안 남김**(세션 한정 — 사용자 요청).
