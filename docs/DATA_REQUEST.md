## 0. 모델 구조와 데이터의 관계 (요청 근거의 뼈대)
납품 모델 = **맨홀-노드 하수 라우팅 GNN** (관악구):
- **노드 = 맨홀**(관악 12,184, GIS 보유) — 관측은 수위계가 붙은 맨홀만(현재 하수 13)
- **엣지 = 상류→하류 유향 관거**(관저고 기반 흐름방향, GIS 보유)
- **노드 입력 = 유입**(강우 × 집수면적 + 빗물받이) · **학습 정답 = 실제 침수 라벨**

**골격은 개념이 아니라 이미 구축·검증됨**: KICT 관악 관망 GIS의 공식 연결(시작맨홀→끝맨홀)로 **노드 13,272·엣지 16,059** 그래프 생성 + 공식 접속관수로 검증(차수 정확 80%·±1 91%·공간정합 77%) + 공식경사·소유역(집수단위 40) 부착. 비어있는 건 입력·정답뿐.

→ **빈 곳은 ①학습 정답 ②노드 입력(강우) ③노드 입력(집수면적) ④관측 노드(신설 센서).** 아래 요청이 각 구성요소를 채운다. (그림: `09_roadmap.png`(구성요소)·`10_manhole_graph.png`(실제 그래프))

## 배경 (왜 현 데이터로 안 되는가 — 관악·여름 기준)
- 도로 침수 라벨: 신뢰 양성이 **조원로 1지점·24건**, 불균형 1:1,208 (봉천동 가짜 1,511건은 겨울 아티팩트)
- 관악 침수 = 하수 **만관(도달 0%)** 도, **위험수위 도달(0.064%, 극소)** 도 아니고, 하천 역류도 아님(침수 90%가 도림천 평상시) → **국지 호우 표면류**
- 강우+이력 모델이 persistence를 CSI로 못 넘음(관악 GNN 0.72 vs 0.17) = 리드타임 부재

### ② 격자(레이더형) 강우  〔모델: 노드 입력(강우) · 리드타임의 본체〕
| | |
|---|---|
| 요청 | 이동하는 강우장을 담은 **격자 강수자료** — 기상청 레이더 합성(HSR 등)·KICT 보정 강수장 등 가용한 것이면 됨 |
| 명세(기능 요건) | ▸점이 아닌 **공간 격자** ▸관악을 해상하기 충분한 해상도(예: **1km 이하**) ▸시간 **5~10분급** ▸상류 포함 영역(이동 관측) ▸2024~현재 ▸좌표계/격자ID/강수단위/결측코드. **※기상청 HSR(500m·5분)은 충족 *예시*이지 필수 규격 아님** — 동등한 격자자료면 가능 |
| 사유 | **리드타임은 모델이 만드는 게 아니라 "미래를 담은 입력"에서 나옴.** 두 시계 중 **①기상 시계**(비가 아직 관악에 안 옴)만 10~60분 선행 제공=레이더. 관악 **②수문 시계**(유출 지연)는 ~0(강우-수위 동시반응, persistence 0.72≫GNN 0.17)이라 그래프만으론 몇 분뿐 → **레이더가 리드타임의 본체**. 점 강우는 리드타임 0. |
| 한계(정직) | 관악 침수형=국지 대류 호우는 셀이 이동뿐 아니라 생성·소멸 → nowcasting이 가장 약한 유형. 리드타임 천장 보수적 10~30분(onset CSI 0.06~0.09가 기준선). |
| 구조 | 레이더 강우장 → nowcast → 예측 강우장 → ×집수면적 → 맨홀 유입 → 유향 GNN (2단 분리: 강우예측 단·침수라우팅 단 독립 검증) |
| 증거 | 보고서 §19 그림 28(관악 GNN) · §17 그림 26(onset) · 상세 `docs/LEADTIME_AND_RAIN_FORECAST.md` |

### ③ DEM (수치표고) — 또는 KICT 하수관망 수리모델(SWMM)  〔모델: 노드 입력(집수면적)〕
| | |
|---|---|
| 요청 | 고해상 DEM/LiDAR **또는** KICT 보유 하수관망 수리모델(SWMM 등, 소배수구역·맨홀별 집수면적 포함) — 둘 중 하나 |
| 명세 | DEM 1m급(또는 가용 최고해상) · 또는 SWMM .inp(소배수구역·집수면적 포함) |
| 사유 | 모델 노드 입력 = 맨홀 유입량 = **강우 × 집수면적**(합리식 Q=C·i·A). 강우(②)만으론 유입 *부피*가 안 나오고, 면적이 곱해져야 "그 맨홀에 실제 들어오는 물의 양"이 됨. 같은 비라도 집수면적 큰 맨홀이 먼저 위험수위·월류 → 모델이 공간 위험을 구분하려면 면적 가중 입력 필수. **GIS 소유역은 40개로 coarse(이미 활용)** → 맨홀별 정밀 집수면적은 **DEM(흐름누적·저지대)에서 도출**하거나 **SWMM이 있으면 그대로 사용**. |
